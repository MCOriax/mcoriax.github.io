How to level up

Every level takes 120 EXP, and the class caps at level 100. Earn experience by performing the actions below — each is configured in this profession's YAML file. Ranges such as a~b award a random amount between a and b; numeric conditions such as 1-10 match the amount involved (for example damage dealt or survived).

ActionHow it triggersEXP
craft_itemCraft an item2 EXP
smelt_furnaceExtract a smelted item from a furnace3 EXP
brew_potionCollect a brewed potion from a brewing stand12 EXP
sculpt_chiseled_blockCut a chiseled block variant on a stonecutter4~6 EXP
use_smithing_templateTake a result out of a smithing table10~15 EXP

These values come from the bundled professions/crafter.yml. Server owners can edit them or add entirely new professions — see Professions & EXP.
