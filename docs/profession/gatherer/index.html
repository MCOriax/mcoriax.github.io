How to level up

Every level takes 130 EXP, and the class caps at level 100. Earn experience by performing the actions below — each is configured in this profession's YAML file. Ranges such as a~b award a random amount between a and b; numeric conditions such as 1-10 match the amount involved (for example damage dealt or survived).

ActionHow it triggersEXP
strip_logStrip a log with an axe (once per location)2~3 EXP
collect_honeyBottle honey from a full beehive or nest5~10 EXP
harvest_sweet_berriesGather berries from a ripe sweet berry bush3~5 EXP

These values come from the bundled professions/gatherer.yml. Server owners can edit them or add entirely new professions — see Professions & EXP.
