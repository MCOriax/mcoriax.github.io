How to level up

Every level takes 100 EXP, and the class caps at level 100. Earn experience by performing the actions below — each is configured in this profession's YAML file. Ranges such as a~b award a random amount between a and b; numeric conditions such as 1-10 match the amount involved (for example damage dealt or survived).

ActionHow it triggersEXP
tame_animalTame an animal25~40 EXP
breed_animalBreed two animals8~12 EXP
leash_entityLeash an entity (rate-limited)5~8 EXP
feed_petFeed one of your own tamed pets (rate-limited)2~4 EXP

These values come from the bundled professions/tamer.yml. Server owners can edit them or add entirely new professions — see Professions & EXP.
