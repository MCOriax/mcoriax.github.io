2.0.0 Latest 2026-07-18 · API namespace

A tidier home for the developer API. Every class now lives under the plugin's own io.github.mcoriax.mcidentity namespace.

Changed Developer API namespace

  • All public API packages moved from io.github.mcoriax.* to io.github.mcoriax.mcidentity.* — for example io.github.mcoriax.common.MCIdentityProvider is now io.github.mcoriax.mcidentity.common.MCIdentityProvider, and the events package is now io.github.mcoriax.mcidentity.bukkit.event.
  • Integrating plugins should update their import statements to the new package paths and bump the dependency version to 2.0.0. The published Maven group id is unchanged (io.github.mcoriax), so only the version and your imports need to change.

Note Server owners

  • Nothing changes for you — this is a drop-in update. Replace the jar and your commands, permissions, and data all keep working exactly as before.
