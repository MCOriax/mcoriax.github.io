Change Log — 1.1.0

Permalink for version 1.1.0.

1.1.0 Latest 2026-07-17 · Universal JAR

One download now runs everywhere. MCIdentity ships as a single universal plugin jar that adapts to whichever server it is installed on.

Changed Universal JAR architecture

  • A single plugin file now seamlessly supports SpigotMC, PaperMC, and Folia automatically without needing separate downloads.
  • The plugin detects your server type on startup and enables the matching scheduling, so Folia's regionalized scheduling is still used automatically when you run Folia.
  • Installation is simpler: drop the one jar into your plugins/ folder — there is no longer a per-platform build to choose from.
